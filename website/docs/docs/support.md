# Support for Eclipse Enclave

Eclipse Enclave is developed as an open source project. If your team is building
on Enclave and needs help beyond the community channels, professional support
and services are available from contributing companies. You can also sponsor the
project directly to fund ongoing development.

## Project sponsoring

Sponsoring keeps the project healthy: it funds core maintenance, security
updates, and new capabilities that benefit every user of Enclave.

  <div className="enclave-box">

If you are interested in either model, contact one of the professional support
providers above to discuss the arrangement that fits your organization.

## Community support

For everyday questions, bug reports, and feature ideas, the project's GitHub
repository is the place to go. Maintainers and community members follow it and
respond there.

- [Report an issue or request a feature](https://github.com/eclipse-enclave/enclave/issues)
- [Browse and join discussions](https://github.com/eclipse-enclave/enclave/discussions)

Please keep in mind that community channels rely on volunteer time. If you need
guaranteed response times, priority fixes, or bespoke work, professional support
is the right fit.
