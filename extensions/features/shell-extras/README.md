# shell-extras

Enhanced shell experience with zsh and oh-my-zsh. Opt-in (disabled by default).

**Priority**: 90

## Packages (apt)

- `zsh`
- `bash-completion`

## Installation

The install script:

1. Installs oh-my-zsh (robbyrussell theme)
2. Configures `~/.zshrc` with PATH, nvm, and terminal size handling
3. Sets up direnv hooks in `feature-entrypoint.d/setup.sh`
