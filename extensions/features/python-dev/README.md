# python-dev

Python development tools installed via [uv](https://github.com/astral-sh/uv).
Enabled by default.

**Priority**: 70

## Installation

Installs `uv` if not already present, then uses `uv tool install` for each tool:

| Tool | Purpose |
|------|---------|
| `black` | Code formatter |
| `ruff` | Linter and formatter |
| `mypy` | Static type checker |
| `pytest` | Test framework |
| `ipython` | Interactive Python shell |
| `poetry` | Dependency management |
| `pipenv` | Dependency management |
