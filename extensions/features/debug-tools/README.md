# debug-tools

Low-level debugging and tracing tools. Opt-in (disabled by default).

**Priority**: 80

## Packages

| Package | Purpose |
|---------|---------|
| `gdb` | GNU debugger |
| `ltrace` | Library call tracer |
| `strace` | System call tracer |
| `lsof` | List open files |
| `tcpdump` | Network packet capture |
